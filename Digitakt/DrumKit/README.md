# Workflow

1. Choose the kick (Track 1)
	- Start with a solid kick sample.
	- Tune it so it fits your song.
	- Use the filter to remove unnecessary high frequencies.
	- Adjust the amp envelope so it's as short or long as needed.
1. Choose the snare (Track 2)
	- Find a snare that contrasts with the kick.
	- Add a little overdrive if it needs more punch.
	- Tune it slightly up or down to fit.
1. Add a clap (Track 3)
	- Layer with the snare or use it separately.
	- Use sample start to change the attack.
1. Closed hi-hat (Track 4)
	- Short decay.
	- High-pass filter if needed.
	- Slight velocity variation makes it feel more natural.
1. Open hi-hat (Track 5)
	- Longer decay.
	- Set it to choke with the closed hat if desired.
1. Percussion (Tracks 6–7)
	- Add toms, rimshots, shakers, or unusual percussion.
	- Pan them slightly for width.
1. Effects (Track 8 or remaining tracks)
	- Cymbals, crashes, noise hits, vocal chops, or FX.

# Kit shape

- Match the levels before adding effects.
- Add a little overdrive where needed.
- Use the filter on every sound to remove frequencies you don't need.
- Add send reverb and delay sparingly.
- Save the sounds once you're happy.

# Digitakt features

- Parameter locks to change pitch, filter, or sample start per step.
- Conditional trigs for variation.
- Micro-timing for groove.
- Retrigs for rolls.
- LFOs for subtle movement on hats or percussion.

# Common track layout

| Track | Instrument        |
| ------| ----------------- |
| T1    | Kick              |
| T2    | Snare             |
| T3    | Clap              |
| T4    | Close Hat         |
| T5    | Open Hat          |
| T6    | Percussion        |
| T7    | Tom or Shaker     |
| T8    | Crash, Ride or FX |


This gives you a versatile kit that can cover most electronic genres while leaving plenty of room for Digitakt's sequencing tricks.

# Starting a new Project

1. Create a new project
	1. Press `FUNC + SETTINGS` (or open the Project menu, depending on your Digitakt model).
	1. Go to Projects.
	1. Select New Project.
	1. Give it a name.
	1. Save it.
1. Set the project tempo
	1. Press TEMPO.
	1. Set the BPM.
	1. Set the time signature if needed.
1. Create a new pattern
	1. Select Bank A.
	1. Choose Pattern A01 (or any empty pattern).
	1. If it's not empty, clear it:
	1. FUNC + CLEAR (or use the Pattern menu to clear the pattern).
1. Load samples
	For each audio track:
	1. Press the desired Track button (T1–T8).
	1. Open the Sample Browser.
	1. Browse your sample folders.
	1. Assign a sample to the track.

	A common layout is:
	- T1 – Kick
	- T2 – Snare
	- T3 – Clap
	- T4 – Closed Hat
	- T5 – Open Hat
	- T6 – Percussion
	- T7 – Bass or Tom
	- T8 – FX or Crash
1. Set the pattern length
	- Open the Scale Setup.
	- Choose 16, 32, or 64 steps depending on the groove you want.
1. Build your beat
	Record or place trigs:
	- Kick first.
	- Then snare.
	- Then hats.
	- Finally percussion and fills.
1. Shape each sound
	For every track, adjust:
	- Sample Start/End
	- Pitch
	- Filter
	- Amplitude envelope
	- Overdrive
	- LFO (optional)
1. Add effects
	Set:
	- Reverb send
	- Delay send
	- Master compressor (if desired)
1. Save everything
	- Save the pattern.
	- Save the project.
	- Save any sounds you want to reuse later.

This gives you a clean, organized project ready for sequencing and further sound design.
